Design Notes

Dark glass panels, cold-blue edges, and bioluminescent green highlights match the Squidley V2 family without copying the exact layout.

Breathing animation is subtle by default so live run cards can feel active without turning the whole page into motion soup.

Typography uses IBM Plex Sans plus JetBrains Mono to keep the same operational tone as the rest of the lab.
